# Graphisme Ensemble — Contexte projet

Site web de l'association **Graphisme Ensemble**, association loi 1901 basée à La Réunion.
Déployé sur Netlify, repo GitHub : https://github.com/emeric-H974/graphisme-ensemble

## Stack technique
- Site statique mono-fichier : `index.html`
- Navigation SPA en vanilla JS (pas de framework)
- Formulaires via Netlify Forms (contact + newsletter)
- Logo : `project/uploads/Graphisme.png`
- Polices Google Fonts : Archivo + Caveat
